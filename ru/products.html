Основные характеристики

  • Беспроводная портативная конструкция для гибкости на объекте
  • Работает с гнездом 16-32mm и арматурой 16-40mm
  • Макс. крутящий момент: 1200N
  • Скорость двигателя: 3000 r/min
  • 60-90 мин время работы на заряд
  • Компактный и лёгкий, 8,5кг

Катушка вязальной проволоки 0,8мм

Премиальные катушки вязальной проволоки 0,8мм для арматуры, предназначенные для аккумуляторных вязальных машин (серия RT528 / RT545 / RT558). Доступны в оцинкованном, чёрном отожжённом, с ПВХ-покрытием и нержавеющем вариантах. 100м на катушку.

Основные характеристики

  • Предназначен для машин для вязки арматуры серии RT
  • Одиночная проволока 0,8mm (21 Ga) — 100m на катушку
  • Высокая прочность на разрыв: 480-520 MPa
  • Несколько вариантов: Оцинкованный / чёрный отожжённый / с PVC-покрытием / нержавеющая сталь
  • Материал Q195 для стабильной надёжной работы
